## Как скачать yarn?

npx corepack enable


# Для проверки версии yarn
yarn --version

# Для установки пакета

yarn

## Для запуска проекта

yarn dev

### Структура компонента должна выгядеть примерно так

Component.tsx
Component.module.scss
types.ts
